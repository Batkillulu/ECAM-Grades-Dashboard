# Privacy Policy

ECAM Grades Dashboard runs entirely client-side as a browser userscript.

## Data Collection
No personal data is collected.
No data is transmitted to any external server.
No tracking, analytics, or third-party services are used.

## Data Processing
The script only processes academic data already visible to the user on the ECAM intranet.
All computations are performed locally in the browser.

## Local Storage
Some user preferences and simulated grades may be stored locally in the browser
using localStorage. This data never leaves the user’s device.
